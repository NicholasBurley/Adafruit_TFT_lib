◆ setRotationTFT()

void setRotationTFT ( uint8_t  r,
TFTVars *  var 
)

Sets the rotation of the screen. The screen can have four different rotations. Updates the width and height according to rotation. Rotation 0 has 0 for x and y at the bottom left corner. Where x is height and y is width. Each subsequent rotation turns the axes clock wise. Where rotation 1 has the 0 at the upper left corner and the x-axis as width and y-axis as height.

Parameters
rrotation can be from 0 to 3.
varpointer to TFTVars structure
