7 //* Target Device : ATmega324A
8 //* Description:
9 //* Contains all the methods used to communicate with the
10 //* Adafruit touch screen. This file specifically contains
11 //* methods to send data or commands to the touch screen. All methods
12 //* use SPI to send data.
13 //* Notes:
14 //* Enable (or disable) Code Page 437-compatible char set.
15 //* There was an error in glcdfont.c for the longest time -- one character
16 //* (#176, the 'light shade' block) was missing -- this threw off the index
17 //* of every character that followed it. But a TON of code has been written
18 //* with the erroneous character indices. By default, the library uses the
19 //* original 'wrong' behavior and old sketches will still work. Pass 'true'
20 //* to this function to use correct CP437 character values in your code.
21 //*******************************************************************
